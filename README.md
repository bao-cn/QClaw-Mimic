# QClaw-Mimic
QClaw-Mimic is a project that imitates the QClaw website of Tencent. 
QClaw website: https://claw.guanjia.qq.com/

## Run & Build
Run:
```bash
git clone https://github.com/bao-cn/QClaw-Mimic.git
cd QClaw-Mimic
pnpm i
pnpm dev
```
Build:
```bash
pnpm build
```
